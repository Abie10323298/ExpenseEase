# ExpenseEase

**ExpenseEase** is a mobile budgeting application designed to help users take control of their personal finances. The app allows users to track expenses, plan budgets, set financial goals, and view insightful reports — all in one convenient place.

## 📱 Features

- 🔐 **Secure User Authentication**  
  Sign up and log in securely using encrypted credentials.

- 📊 **Budget Tracking**  
  Create and manage custom budgets for different categories (e.g., Food, Transportation, Bills).

- 🧾 **Expense Management**  
  Record daily expenses with category tags, dates, and descriptions.

- 🤝 **Collaborative Budgeting**  
  Share budgets with family members or roommates to manage shared expenses.

- 🎯 **Debt Payoff Planning**  
  Set debt goals and track your payoff progress.

- 🎮 **Gamification**  
  Earn badges and points for staying on budget and saving money.

- 📈 **Financial Reports**  
  View charts and summaries of your spending habits over time.

## 🛠️ Tech Stack

- **Frontend**: Android (Kotlin/Java)
- **Backend**: Firebase (Authentication, Firestore, Cloud Functions)
- **UI/UX**: Material Design Principles

## 🚀 Getting Started

1. **Clone the repository**
   git clone https://github.com/Abie10323298/ExpenseEase.git

2. **Open in Android Studio**
   - Launch Android Studio.
   - Choose "Open an existing project."
   - Select the cloned `ExpenseEase` folder.

3. **Sync Gradle and Run**
   - Wait for Gradle to finish syncing.
   - Connect an Android device or start an emulator.
   - Click **Run**.
